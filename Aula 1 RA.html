Construa seu Robô


  • Para cabeça, uma bola
  • Para os olhos um sol ou relógio
  • Para o nariz um triângulo
  • Para boca uma fruta (mas nao pode ser banana)
  • Para as orelhas duas bananas
  • Para o cabelo folhagem
  • Para o pescoço uma borboleta
  • Para o corpo um coração
  • Para os braços dois guarda-chuva
  • Para as pernas dois objetos

Compartilhe com sua turma como ficou seu robô!!!
