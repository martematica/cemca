Site acessivel

Nesta aula os alunos vão fazer um site acessivel para deficiente visual, sobre os temas das aulas dos três IFAs a ideia e cada aluno fazer o site sobre um tema de uma das componetes, e depois reunir todos os temas em um unico site, e para finalizar utilizar um leitor de tela para que possa acessar usando os navegadores com as teclas tab, enter e espaço.

Temas de cada aluno das aulas

  • Disco de Newton
  • Leds RGB
  • Sensor utrasonic
  • Plataforma Tinkercad
  • Semáforo acessivel deficiente visual
  • Optica Fisica
  • Como nossos olhos enxergam as cores
  • Como funciona a lentes dos óculos
  • Propiedade da luz
  • Espelhos
  • Fibra Optica
  • Funcionamento dos faróes
  • Daltonismo
  • Platforma Github
  • Plataforma p5.js
  • Linguagens HTML e CSS
  • IA inteligencia artificial
  • Arduino
  • Refração da luz
  • Reflexão da luz

Rública

Tem que ter
  1. Titulo centralizado
  2. Texto explicativo
  3. Imagem ilustrativa
  4. link que leve a outra página
  5. Cor de fundo em contraste com a cor da letra
  6. Um gif
  7. Um video incorporado do tema
